Instructions:


PickMyFood Inventory System is a web application designed for pick-up-food station owners to keep track of their menus.
Using this application you can task menu flows by locations.
first of all, you need to add location information in locations, and then go inventory category and then add menu items information input.
and then you can add item information. thank you.

Home

Welcome message and instruction view.

Locations

Creating location will need to name, location number.

Menu

Creating menu is first step in your items, it will make easy to add item quantities. Once you click "add item" you should see it immediately in you inventory. To remove a menu simply click "remove menu" which is next to the add menu button. **if you remove a menu, all items of that menu will also be removed.

Inventory

Creating and removing items.

Search

User can search menu by using location number. You can view location name, location number, quantity, and history.

Tasks

User can task inventory.


Histories

User can keep track of every change that happens.


Task

User can add and remove task.
